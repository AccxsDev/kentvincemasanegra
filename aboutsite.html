About This Site

Purpose of This Site

Welcome to the personal portfolio and gallery of Kent Vince S. Masanegra. This website is designed to showcase personal information, family, and creative works with interactive and visually engaging features.

Key Features

  • Interactive Profile Section with modals linking to Facebook.
  • Family Showcase with detailed descriptions and social links.
  • Gallery Section with responsive grid, modals, and image download option.
  • Animated Backgrounds including starfield and ripple effects.
  • Sidebar navigation (hamburger menu) for About Me and Gallery pages.
  • Modals for images and profile details without leaving the page.
  • Responsive Design adapting to mobile and desktop screens.
  • Footer with social links and "About This Site" link.

Purpose of Each Element

  • HTML structure for sections, modals, and content layout.
  • CSS for hover effects, glow animations, gradients, and responsive styling.
  • JavaScript for interactive elements: modals, scroll reveal, canvas animations.
  • Static hosting of media like images, gallery thumbnails, and profile pictures.
  • User interactions like hovering, clicking, scrolling animations, and back navigation.

How to Navigate

  1. Click the hamburger menu to open the sidebar and navigate to About Me or Gallery.
  2. Hover over profile/family cards to see pop-up scale and glow effects.
  3. Click profile or family images to open modals with more info and social links.
  4. Click images in the Gallery section to view full size and download them.
  5. Enjoy the starfield and ripple animations in the background while browsing.

Note

This site is purely a demonstration of web development techniques. No personal data is collected or stored.
